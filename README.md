porkroll
======

Copyright:  2015 G2, Inc

License:    2-Clause BSD

Porkroll is a zsh script that takes a set of rules files and generates
deployable Snort shared objects from them.

Requirements
============

* If on a BSD: gmake
* autoconf
* automake
* libtool
* pcre
* zsh
* daq
* libdnet
* jq
* A compiler, preferably clang
* pkg-config
